# Restaurant-Review-Classification
Simple Instructions to Run the App
1) Install the required Python libraries (if not installed):
pip install pillow nltk scikit-learn joblib
2) Open PowerShell (or Terminal) and go to the project folder:
cd "C:\path\to\your\project\folder"
3) press Enter. If it works, your prompt should change to that folder.
4) Now type:
python RestaurantReviewApp.py
